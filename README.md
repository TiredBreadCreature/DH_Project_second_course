Рабочий репозиторий (ура)

Данный репозиторий существует чтобы поддерживать работу сайта https://tiredbreadcreature.github.io/DH_Project_second_course/
Использованные датасеты взяты из Портала открытых данных Правительства Москвы. (https://data.mos.ru/)
Проект выполнен в среде Jupyter и преобразован в сайт через Quarto и VS

Целью проекта являлось составление карт концентрации доступности культуры Москвы, чтобы можно было поразмышлять о причинах тех или иных особенностей и в целом понять, как в Москве работало и работает размещение и сохранение подобных объектов.
